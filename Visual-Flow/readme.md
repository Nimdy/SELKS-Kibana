1. Open Kibana
2. Click Vizualize
3. Click +
4. Click Vega
5. Cut and Paste script
6. Click save
7. Name vizalization
8. Click Dashboard
9. Click SN-FLOW
10. Click edit
11. Click add
12. Search for newly created vizualization
13. Scroll to the bottom of the dashboard and drag it to your liking
14. Click save
15. Overwrite current dashboard
16. Complete
